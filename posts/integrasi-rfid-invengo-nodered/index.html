Integrasi RFID Invengo XC-RF850 dengan Raspberry Pi & Node-RED#

Sebuah Pendekatan IoT Gateway Berbasis Edge Computing#

Abstrak#

RFID (Radio Frequency Identification) merupakan teknologi utama dalam sistem logistik, pergudangan, dan otomasi industri modern. Penelitian ini membahas implementasi integrasi RFID Reader Invengo XC-RF850 dengan Raspberry Pi 3+ menggunakan Node-RED sebagai middleware IoT Gateway. Sistem dirancang untuk menerima data EPC secara real-time melalui komunikasi TCP/IP, melakukan parsing data mentah (raw binary), dan menghasilkan data terstruktur dalam format JSON yang siap diintegrasikan ke sistem tingkat lanjut seperti database, REST API, MQTT broker, dan dashboard monitoring.

Kata Kunci#

RFID, Invengo XC-RF850, Raspberry Pi, Node-RED, IoT Gateway, Edge Computing

1. Pendahuluan#

Perkembangan Internet of Things (IoT) mendorong kebutuhan akan sistem identifikasi otomatis yang cepat, andal, dan terintegrasi. RFID menjadi solusi utama karena kemampuannya membaca identitas objek tanpa kontak fisik dan dengan kecepatan tinggi.

Namun, banyak sistem RFID industri masih bersifat closed system dan sulit diintegrasikan dengan platform IoT modern. Oleh karena itu, penelitian ini mengusulkan pendekatan open-architecture IoT Gateway menggunakan Raspberry Pi dan Node-RED sebagai penghubung antara perangkat RFID industri dan sistem aplikasi.
