Dokumentasi PALASIK

Mesin keputusan keamanan untuk event IoT di sisi edge.

Konsep Inti

PALASIK mengevaluasi setiap event masuk lalu menetapkan keputusan security secara real-time.

  • Trust Engine menilai perilaku perangkat.
  • Policy Engine menjalankan keputusan ALLOW / RESTRICT / DENY.
  • Semua keputusan tercatat sebagai evidence.

Kaitan dengan DEMIT

DEMIT adalah super app runtime; PALASIK adalah aplikasi keamanan yang dijalankan melalui routing DEMIT.

DEMIT

Menangani lifecycle dan routing antar aplikasi.

PALASIK

Menentukan apakah event diproses, dibatasi, atau ditolak.

Quick Start Singkat

demit.yaml

demit:
  apps:
    - palasik

Untuk panduan instalasi penuh, health check, simulasi event, dan monitoring status, buka Dokumentasi Teknis PALASIK.

Acuan Penelitian & Lanjutan

Untuk riset lanjutan PALASIK dan PiTUSK, gunakan acuan metodologi dan indikator berikut:
