Dokumentasi DEMIT

Runtime super-app yang menjadi host untuk PALASIK.

Ringkas

DEMIT menjalankan beberapa aplikasi dari satu proses, dengan routing event yang fleksibel dan lifecycle yang konsisten.

  • Start/Stop lifecycle aplikasi
  • Routing event via route dan nama aplikasi
  • Konfigurasi deklaratif melalui demit.yaml

Kaitan dengan PALASIK

PALASIK adalah aplikasi default yang di-load oleh DEMIT untuk enforcement keamanan.

DEMIT mengatur orkestrasi, PALASIK menjalankan keputusan.

Contoh Konfigurasi Singkat

demit:
  apps:
    - palasik

apps:
  palasik:
    type: palasik
    config_file: config.yaml
