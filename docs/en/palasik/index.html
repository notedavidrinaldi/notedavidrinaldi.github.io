PALASIK Documentation (English)

Security decision engine for IoT events in DEMIT super-app runtime.

Core Concept

PALASIK evaluates every event from edge devices and produces decision outcomes (ALLOW, RESTRICT, DENY, etc.) through trust and policy layers.

  • Trust Engine computes dynamic trust score.
  • Policy Engine selects enforcement action.
  • Decision reasons are logged for auditability.

Relation with DEMIT

DEMIT acts as the app runtime host, and PALASIK is the default security application registered in demit.yaml.

DEMIT Role

Router and lifecycle orchestration for security applications.

PALASIK Role

Per-event decision logic and action execution.

Quick Start

demit.yaml

demit:
  apps:
    - palasik

apps:
  palasik:
    type: palasik
