DEMIT Documentation (English)

Super app runtime host where PALASIK runs as a security module.

What is DEMIT

DEMIT (Digital Monitoring Super App) provides app lifecycle and routing while keeping the core lightweight and extensible.

  • Single-process modular runtime
  • Declarative configuration via demit.yaml
  • Event routing to specific modules

Relation with PALASIK

Current default application inside DEMIT is PALASIK, responsible for security decisions and policy enforcement.

Key relation: DEMIT routes, PALASIK decides and enforces.

Example Configuration

demit:
  apps:
    - palasik

apps:
  palasik:
    type: palasik
    config_file: config.yaml
