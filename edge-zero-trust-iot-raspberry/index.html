Abstrak#

Keamanan pada sistem Internet of Things (IoT) masih menjadi tantangan besar, terutama pada lingkungan dengan perangkat terbatas sumber daya. Pendekatan keamanan tradisional yang mengandalkan perimeter network sudah tidak relevan. Penelitian ini mengusulkan Edge-based Zero Trust IoT Security Framework menggunakan Raspberry Pi 3B sebagai gateway utama, yang menerapkan verifikasi berkelanjutan terhadap setiap perangkat IoT tanpa asumsi kepercayaan awal.

Framework ini dirancang ringan, modular, dan dapat direproduksi untuk keperluan penelitian maupun implementasi industri skala kecil hingga menengah.

1. Pendahuluan#

Pertumbuhan perangkat IoT membawa risiko keamanan yang signifikan, seperti:

  • Perangkat tanpa autentikasi kuat
  • Firmware yang jarang diperbarui
  • Komunikasi tanpa enkripsi

Model Zero Trust Architecture (ZTA) menawarkan pendekatan baru dengan prinsip never trust, always verify. Namun, sebagian besar implementasi Zero Trust masih berfokus pada cloud atau data center. Penelitian ini memindahkan konsep Zero Trust ke edge device menggunakan Raspberry Pi.

3. Konsep Zero Trust di Edge IoT#

Prinsip utama yang diterapkan:

  1. Tidak ada perangkat yang otomatis dipercaya

  2. Setiap koneksi diverifikasi

  3. Trust bersifat dinamis

  4. Keputusan keamanan dibuat di edge

3.1 Trust Score Model#

TrustScore(t) = α · Authentication + β · Fingerprint + γ · Behavior

6. Studi Kasus Implementasi#

  • Framework ini dapat digunakan pada:

  • Sistem RFID di pelabuhan

  • Monitoring industri kecil

  • Smart agriculture

  • Penelitian keamanan jaringan IoT

7. Kontribusi Penelitian#

  • Kontribusi utama dari framework ini adalah:

  • Implementasi Zero Trust di edge low-power

  • Trust berbasis perilaku, bukan hanya identitas

  • Modular dan open research friendly

  • Cocok untuk lingkungan tanpa cloud

8. Peluang Pengembangan Lanjutan#

  • Integrasi Machine Learning untuk anomaly detection

  • Dukungan hardware TPM

  • Federated Trust antar gateway

  • Dataset publik untuk riset IoT Security

10. Kesimpulan#

Penelitian ini menunjukkan bahwa konsep Zero Trust dapat diterapkan secara efektif pada edge device seperti Raspberry Pi 3B. Framework yang diusulkan memberikan pendekatan keamanan yang fleksibel, ringan, dan sesuai untuk penelitian serta implementasi nyata.
