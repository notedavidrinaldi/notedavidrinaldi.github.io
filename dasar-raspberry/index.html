1. Hardware Deep-Dive#

Raspberry Pi Dasar ðŸŸ¦#

Komputer mini berbasis ARM untuk edukasi, hobi, dan penelitian.

  • Tipe 4B: CPU Cortex-A72, RAM 2-8GB (Populer untuk heavy processing).
  • Tipe 3B+: CPU Cortex-A53, RAM 1GB (Stabilitas prototyping).
  • Zero 2 W: Sangat kecil, hemat energi untuk node sederhana.

2. Instalasi & Konfigurasi#

Alur Setup Raspberry Pi ðŸŸ¦#

  1. Download Image: Via Raspberry Pi Imager.
  2. Flash microSD: Minimal 16GB Class 10.
  3. Boot & Config: Setup Hostname & SSH.
  4. Services: Install Node-RED & MQTT Broker.

Alur Setup Siemens IoT 2050 ðŸŸ©#

  2. Flash Storage: Flash ke eMMC (Advanced) atau microSD.
  3. First Login: Default user root (tanpa password).
  4. IOT Setup: Jalankan perintah iot2050setup untuk konfigurasi I/O.

4. Security Checklist (Hardening)#

  1. Ganti Password: Jangan pernah biarkan password default aktif.
  2. Disable Root SSH: Gunakan PermitRootLogin no di sshd_config.
  3. UFW Firewall: Tutup semua port kecuali 22, 1883, dan 8080.
  4. SSH Keys: Gunakan autentikasi kunci publik (RSA/ED25519).

5. Linux CLI Cheatsheet#

  • ip addr: Mengecek alamat IP perangkat.
  • nmcli: Konfigurasi jaringan via command line.
  • vcgencmd measure_temp: Monitor suhu CPU (khusus RPi).
  • journalctl -u node-red: Debugging log service Node-RED.

Dibuat untuk keperluan riset IIoT. Selalu gunakan hardware original dan catu daya yang stabil untuk validitas data riset Anda.
