Raspberry Pi adalah komputer mini berbasis ARM untuk edukasi, hobi, dan penelitian. Memiliki port GPIO untuk sensor dan aktuator.

2.3 HDMI vs Mini HDMI#

  • HDMI penuh: langsung ke monitor
  • Mini HDMI: perlu adaptor, hemat ruang

2.4 OS yang didukung#

  • Raspberry Pi OS / Raspbian
  • Ubuntu Server / Desktop
  • DietPi, Kali Linux ARM

2.5 Cara instal OS ðŸŸ¦#

Persiapan#

  1. Download image OS
  2. Flash ke microSD (balenaEtcher / dd)

Boot & konfigurasi awal#

  1. Masukkan microSD, sambungkan catu daya
  2. Konfigurasi jaringan & SSH
  3. Update sistem:
sudo apt update && sudo apt upgrade -y

Install Node-RED#

sudo apt install nodejs npm -y
sudo npm install -g --unsafe-perm node-red
