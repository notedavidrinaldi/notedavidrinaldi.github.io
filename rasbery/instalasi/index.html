Instalasi Rasbery

Langkah praktis dari nol sampai service jalan.

Pra-install

  • Siapkan Pi 4/5, PSU stabil, kabel Ethernet stabil.
  • Siapkan storage dan backup.
  • Pilih OS Raspberry Pi OS Lite untuk edge.

Flash & akses awal

sudo apt update && sudo apt upgrade -y
sudo apt install -y ca-certificates ufw python3-venv git

Hardening minimum

  • Ubah password bawaan user
  • Nonaktifkan login SSH root
  • Batas port dan policy firewall

Checklist go-live

Cek network, clock, timezone, service daemon, dan log startup.
